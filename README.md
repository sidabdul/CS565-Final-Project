# CS565-Final-Project

Application Description

A web-based Todo list app to manage daily
tasks.

Chosen because I use It daily in personal life on
the iPhone and utilizes fullstack development.

App will use frontend, backend, and API
together.

Common real-world example.

To run this ToDo app:

1. Go to the root directory of this project
2. Run npm install concurrently --save-dev
   npm run dev
3. Go to http://localhost:5199/
